# Portfolio Projects

## Planned Projects

### 1. Azure Secure VM Web Server Project
Deploy and secure a Linux virtual machine in Azure, configure Network Security Group rules, connect through SSH, install Nginx, and host a custom web page.

**Skills:** Azure, Linux, SSH, NSGs, Nginx, virtual machines, troubleshooting

### 2. Terraform Azure Infrastructure Deployment
Use Terraform to automate the deployment of Azure infrastructure, including a virtual machine, networking, and supporting resources.

**Skills:** Terraform, Infrastructure as Code, Azure, automation, cloud deployment

### 3. IAM and Least Privilege Security Lab
Create a beginner IAM security project focused on user roles, least privilege, MFA, and secure access design.

**Skills:** IAM, least privilege, MFA, access control, cloud security, security best practices

### 4. Linux and Networking Troubleshooting Lab
Practice common troubleshooting tasks involving connectivity, ports, DNS, firewalls, and service issues on Linux systems.

**Skills:** Linux, networking, troubleshooting, DNS, ports, command line

### 5. GitHub Actions CI/CD Pipeline Lab
Build a simple workflow that automatically runs when code is pushed to GitHub to demonstrate basic CI/CD and automation skills.

**Skills:** GitHub Actions, CI/CD, automation, DevOps basics, YAML

### 6. Docker Container Deployment Lab
Containerize a simple beginner app and run it locally to understand images, containers, and basic deployment workflows.

**Skills:** Docker, containers, Linux, application deployment, basic DevOps
