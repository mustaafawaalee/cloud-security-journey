# IAM and Least Privilege Checklist

## IAM Security Checklist
- [ ] Users should only have access needed for their role
- [ ] MFA should be enabled for important accounts
- [ ] Admin access should be limited
- [ ] Permissions should be reviewed regularly
- [ ] Shared accounts should be avoided
- [ ] Unused accounts should be removed
- [ ] Logging and monitoring should be enabled
- [ ] Sensitive resources should have tighter access controls
