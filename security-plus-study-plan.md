# Security+ Study Plan

## Goal
Pass the CompTIA Security+ exam and build a strong cybersecurity foundation.

## Focus Areas
- General Security Concepts
- Threats, Vulnerabilities, and Mitigations
- Security Architecture
- Security Operations
- Security Program Management and Oversight

## Weekly Plan
- Practice exams
- Review weak areas
- Take notes on missed questions
- Study one domain at a time
- Track progress every week

## Current Priority
Build consistency and improve understanding, not just memorization.
