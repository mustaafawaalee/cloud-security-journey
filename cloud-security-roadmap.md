# Cloud Security Roadmap

## My Goal
Build the skills, projects, and certifications needed to move into cloud security.

## Phase 1: Foundation
- Pass CompTIA Security+
- Learn GitHub basics
- Improve Linux basics
- Strengthen networking fundamentals
- Learn basic Python or PowerShell

## Phase 2: Cloud Fundamentals
- Learn core cloud concepts
- Understand shared responsibility model
- Learn IAM, MFA, logging, and monitoring
- Study AWS or Azure fundamentals

## Phase 3: Hands-On Security Projects
- IAM least privilege project
- Cloud security checklist project
- Logging and monitoring notes
- Security documentation and labs

## Phase 4: Portfolio and Job Readiness
- Upload projects to GitHub
- Improve README and documentation
- Build a professional cybersecurity portfolio
- Tailor resume toward cloud security roles

## Weekly Focus
- Study
- Practice
- Build
- Document
