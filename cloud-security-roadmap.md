# Cloud Security Roadmap

## My Goal
Build the technical skills, hands-on projects, and portfolio needed to move into a cloud engineering or cloud security role.

## Phase 1: Foundation
- Pass CompTIA Security+
- Learn GitHub basics
- Improve Linux fundamentals
- Strengthen networking fundamentals
- Learn basic Python or PowerShell
- Practice documenting technical work clearly

## Phase 2: Cloud Fundamentals
- Learn core cloud concepts
- Understand the shared responsibility model
- Learn IAM, MFA, logging, and monitoring
- Study Azure fundamentals
- Learn how virtual machines, storage, networking, and security groups work
- Understand basic cloud cost awareness and resource cleanup

## Phase 3: Hands-On Cloud Projects
- Azure VM Web Server Project
- IAM and Least Privilege Lab
- Linux and Networking Troubleshooting Lab
- Cloud security checklist project
- Logging and monitoring lab
- Security documentation and screenshots

## Phase 4: Automation and DevOps Basics
- Learn Terraform basics
- Build Terraform Azure VM Deployment
- Learn Docker basics
- Build a beginner Docker app project
- Learn GitHub Actions basics
- Build a CI/CD lab

## Phase 5: Portfolio and Job Readiness
- Upload projects to GitHub
- Improve README and project documentation
- Build a professional cloud/cloud security portfolio
- Tailor resume toward cloud and cloud security roles
- Practice explaining projects like real interview examples

## Weekly Focus
- Study
- Practice
- Build
- Document
- Review progress
